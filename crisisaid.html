Transparency & Trust Guardrails

To prevent AI hallucination and misinformation, this system uses a deterministic approach. The LLM strictly classifies the unstructured input (intent extraction). The output is then mapped exclusively against a human-verified, static database of programs. Verification dates indicate when human auditors last confirmed the program's active status.

System Architecture
USAII Challenge Alignment

Human-in-the-Loop

The AI does not make eligibility or approval decisions. It acts as an intelligent router, mapping unstructured crisis narratives to verified program categories. A human case worker at the organization makes the final decision.

Data Flow Pipeline

1. Input

User provides unstructured plain-language text describing their situation.

2. LLM Classification

Google Gemini processes the text, outputting ONLY structured JSON identifying core needs and urgency.

3. Deterministic Matching

The JSON is matched against a human-audited, verified local database to eliminate hallucination risks.

4. Output

User receives ranked, verified contact points and takes agency to call the programs.
